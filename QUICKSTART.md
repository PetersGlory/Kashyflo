# Kashyflo Quick Start Guide

**Get started with smart budgeting in 2 minutes!**

## Installation

### Local Development
```bash
# Install dependencies
pnpm install

# Start dev server
pnpm dev

# Open http://localhost:3000
```

### Deploy to Vercel
```bash
# Push to GitHub, then:
# 1. Go to vercel.com/new
# 2. Import your GitHub repo
# 3. Deploy (no environment variables needed!)
```

---

## First 3 Steps

### 1️⃣ Add Your Salary
1. Click **Settings** (⚙️) icon
2. Enter monthly salary amount
3. Click **"Add Salary"**

### 2️⃣ Review Budget Split
The default budget allocates your salary as:
- Rent: 20%
- Food: 30%
- Savings: 15%
- Buffer: 10%
- Entertainment: 10%
- Transport: 10%
- Other: 5%

**Skip this if you like defaults, or customize percentages before saving.**

### 3️⃣ Add Your First Expense
1. Click the **+** button (bottom right)
2. Tap a category button
3. Enter amount
4. Click **"Save Expense"**

**Done!** Your dashboard now shows remaining budget per category.

## Smart Features to Know

✓ **Real-time Updates**: Budget updates instantly when you add expenses
✓ **Color Indicators**: Green (plenty left) → Yellow (careful) → Red (low funds)
✓ **Offline Works**: App functions 100% without internet
✓ **Mobile App**: Install on phone (Chrome: tap ⋮ → "Install app")
✓ **Auto Reset**: Budget resets monthly on the 1st

---

## Tips for Success

1. **Add expenses immediately** - Track as you spend, not later
2. **Check daily** - Review remaining budget each day
3. **Adjust percentages** - If a category runs out early, adjust next month
4. **Be honest** - Include all purchases, big and small
5. **Backup data** - Save screenshots or export data regularly

---

## Troubleshooting

| Issue | Fix |
|-------|-----|
| "No salary added yet" | Go to Settings and add your monthly amount |
| Allocations won't save | Check that percentages total exactly 100% |
| Data lost | Never clear browser data - it deletes the local database |
| Offline not working | First visit must be online, then it caches |

## What Gets Tracked

- **Expenses**: Every ₦ you spend (date, amount, category, description)
- **Salary**: Your monthly income and allocation percentages
- **Remaining**: Auto-calculated budget per category

**What's NOT tracked**: Your server sends nothing, data stays on your device!
